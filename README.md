# Personal Portfolio Website

A personal portfolio website built with HTML5 and CSS, showcasing my skills, background, and contact information.

## Project Structure

```
Portfolio/
├── index.html        # Main portfolio page
├── images/           # Image assets
│   └── profile.jpg   # Profile picture
├── requirements.txt  # Project requirements
└── README.md         # This file
```

## Sections

| Section   | Description                                       |
| --------- | ------------------------------------------------- |
| **About** | Personal introduction with profile picture         |
| **Skills**| Front-end, back-end, and tool proficiencies listed |
| **Contact**| Form with input validation and social links       |

## Features

- Semantic HTML5 tags (`header`, `nav`, `main`, `section`, `footer`)
- Internal anchor navigation between sections
- Contact form with HTML validation attributes (`required`, `minlength`, `maxlength`, `type="email"`)
- Responsive layout using CSS Flexbox and Grid
- Accessible images with descriptive `alt` text
- External links to GitHub and LinkedIn (open in new tab)
- Sticky navigation header

## How to View

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.

## Technologies Used

- HTML5
- CSS3
