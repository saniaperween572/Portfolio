# Portfolio Website - Requirements

## Tools & Software
- VS Code (code editor)
- Any modern web browser (Chrome, Firefox, Edge)
- Git & GitHub (version control & hosting)

## Languages & Technologies
- HTML5 (semantic markup)
- CSS3 (external stylesheet)
- JavaScript ES6+ (external script)

## CSS Features Used
- CSS Custom Properties (variables)
- Flexbox & CSS Grid layouts
- @keyframes animations (loader, particles, morphing shapes, scroll reveal)
- Gradient backgrounds & text
- backdrop-filter (glassmorphism header)
- Media queries (responsive design)
- prefers-reduced-motion (accessibility)
- Transitions & transforms
- ::before / ::after pseudo-elements
- Dark mode theme (CSS variable overrides)

## JavaScript Features Used
- DOM manipulation (getElementById, querySelector, createElement)
- Event listeners (click, blur, input, change, submit, scroll, keydown)
- IntersectionObserver API (scroll reveal, active nav, stat counter)
- localStorage API (dark mode preference persistence)
- Form validation with custom inline error messages
- Toast notification system
- Typing effect animation
- Debounce function
- IIFE (Immediately Invoked Function Expressions)
- ES6+ features (const, let, arrow functions, template literals)

## Fonts (CDN)
- Google Fonts: Inter, Playfair Display

## Browser Compatibility
- Chrome 80+
- Firefox 75+
- Edge 80+
- Safari 14+
